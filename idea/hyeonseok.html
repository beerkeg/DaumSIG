DaumSIG App Idea

신현석, from Mozilla Korea Community, at Daum Communications, 2013. 9. 12

Background

  • 장보기 앱은 많지만 내 입맛에 맞는 앱은 없다.
  • 많은 장보기 앱이 과도하고 불편한 입력을 요구한다.
    • 간편하고 빠른 입력
    • 쉬운 업데이트
  • 주로 집과 마트에서 쓰게 된다.

Key Feature

  • 입력은 사진 촬영으로
    • 시간남으면 이름도 입력
    • 빈껍데기만 남은 경우도 가능!
  • 구매시 수량 업데이트
    • 수량과 재구매 기간을 고려하여 사야할 물건 추천
  • 집인지 마트인지 판단해서 초기화면 선택해줌

Screen Flow

  • 첫화면
    • 제품 등록(사진 찍기)
      • 다들 아는 기능(생략)
    • 구매할 목록 보기
      • 남은 수량이 적은 || 구매 주기가 다가온 순으로 정렬
      • 구매시 수량입력하면 다음 구매 주기 계산해서 재 정렬
      • 구매주기가 돌아왔는데 이미 있는 경우는 수량 입력

Web API in this App

  • Camera API
  • Local Storage
  • GeoLocation
